# Multi-Agent Verification

Cross-validate critical operations with 2-3 agents.

## When It Triggers

- Complexity >= 15
- Critical keywords (payment, security, auth)
- User --verify flag

## How It Works

- Spawns 2-3 agents in parallel
- Different strategies
- Consensus analysis
- User chooses if disagreements
