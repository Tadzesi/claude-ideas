# /prompt-article & /prompt-article-readme

Interactive wizards for content creation and documentation generation.

## /prompt-article

Generate articles for multiple platforms (LinkedIn, Medium, Dev.to).

### Features
- 8 article types
- Multi-platform output
- Interactive wizard
- Slovak/English support

### Usage
```bash
/prompt-article [topic]
```

## /prompt-article-readme

Generate professional README.md by analyzing your project.

### Features
- Automatic project analysis
- Tech stack detection
- 3 style levels
- Handles existing README

### Usage
```bash
/prompt-article-readme
```
