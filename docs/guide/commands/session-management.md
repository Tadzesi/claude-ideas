# Session Management

Zero information loss between sessions with `/session-start` and `/session-end`.

## /session-start

Load comprehensive context from previous sessions.

### Features
- Loads all sessions
- Aggregates cumulative context
- Highlights active work
- Presents pending items

### Usage
```bash
/session-start
```

## /session-end

Save comprehensive session context.

### Features
- 10-section comprehensive capture
- Decisions, code changes, features
- Problems solved, insights
- Active work and next steps

### Usage
```bash
/session-end
```
