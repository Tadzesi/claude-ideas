## üìù Notes

**Merge Strategy:**
- New findings are appended to existing sections
- Conflicting information is flagged with [CONFLICT] marker
- User resolves conflicts manually
- Auto-summarization triggers at 10 MB file size

**Cross-Session Retention:**
- All knowledge persists indefinitely (90-day default, configurable)
- Git-tracked for version history
- Backed up with project files

**Integration Points:**
- Lead Agent reads this file before research planning
- Result Aggregator updates this file after research completion
- Citation Index cross-references findings
- Architectural Context supplements with deeper analysis
- Learning System uses this to improve smart defaults
