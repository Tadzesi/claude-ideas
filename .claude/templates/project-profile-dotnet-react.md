- Base URL: /api/

## Code Conventions

- Async/await everywhere — no .Result/.Wait()
- CancellationToken on all IO methods
- ILogger<T> for logging
- Nullable reference types enabled
- Records for DTOs
- No hardcoded connection strings — use IConfiguration

## Known Gotchas

- React Router basename must be /[stack-name]/ or navigation breaks
- Vite base must be /[stack-name]/ in production build
- CORS must include both lab463.com and local IP:PORT
- .NET publish must target linux-x64 (server has no .NET SDK)
- SCP uploads to publish/ and frontend/ in stack directory
